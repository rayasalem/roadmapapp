============================================
  إعداد Render – أوامر جاهزة من المستودع
============================================

الملف render.yaml يحدد الإعداد. إن لم يُطبّق تلقائياً، اضبط يدوياً:

  Build Command:
    npm install && npm run build

  Start Command (الافتراضي يعمل الآن):
    node expo/AppEntry.js

  Node: 20 (من .nvmrc)

بعد البناء، الملف expo/AppEntry.js يشغّل الخادم الذي يخدم مجلد dist.
